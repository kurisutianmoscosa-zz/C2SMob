SAM-1™ Use

Connect SAM-1™

To use the Sensorex SAM-1™ meter, attach the module to the headphone jack of your Apple or Android device. Then connect a Sensorex Smart Sensor such as the Sensorex S2000C to the Smart connector on the bottom of the SAM-1™(opposite of the cable and audio connector).

Remove the supplied soaker bottle or soaker cap that keeps the sensing surfaces wet, but consider saving for future storage of the sensor

To verify accuracy of reading, follow calibration guide in the information section.

Taking measurements

To take a reading, dip the sensor into the solution to be measured. The display will show the measurement value, which will stabilize over time.

Recording readings

Tap the record measurement icon ("+") at the top of the display to take a recording of the measured value. This will let you save the reading to the history database. If your device is "location aware", the coordinates where the reading was taken will be stored with it. The air and solution temperatures are also recorded.

When recording a reading, you have the option to add a site location.  Site tags are used to group readings together for history displays. You can use any descriptive text for the site name.

There are optional fields available when entering readings: Comments and User. These fields can be used to take general notes, information about the sensor being used, and details of the person taking the measurements.

Recorded readings can be emailed through from the history screen. You can select individual or multiple readings.

Temperature

The activity of the H+ ions and Conductivity varies with temperature, since pH and Conductivity are temperature sensitive parameters, temperature readings are used to correct the value.  Solution compensation for temperature must be made if highly accurate readings are required for applications with fluctuating temperatures.

The user has three options for temperature compensation, Manual, Air or Sensor.  Select Settings in the upper right-hand corner of the display to select which value will be used for correction, and the units to use for display.

  • "Manual" temperature allows the user to manually input a value or use the default of 25°C when the sensor being used does not have a integrated temperature sensor.
  • "Air" temperature uses a built in sensor within the SAM-1™ module and is available only when using a non-temperature compensated sensor.
  • "Sensor" is automatically selected when the sensor has a built-in temperature sensor to provide the solution temperature.

Solution temperature compensation is accomplished via the smart connector.  The S2900C is a 3-in-1 pH sensor that is ideal for compensating via solution temperature.
